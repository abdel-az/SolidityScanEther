# VulneraScan
AI-powered smart contract vulnerability scanner using Solidity and machine learning.

## Getting Started

### Prerequisites
- Node.js >= 18
- npm or yarn
- Python >= 3.10 (if backend uses Python)
- Docker (optional, for containerized setup)

### Installation
```bash
git clone https://github.com/your-username/VulneraScan.git
cd VulneraScan
npm install
API_KEY=your_api_key_here
ETH_NODE_URL=https://mainnet.infura.io/v3/your_project_id
